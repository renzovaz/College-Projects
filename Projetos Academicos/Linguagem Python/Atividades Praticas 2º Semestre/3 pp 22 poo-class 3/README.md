- Resolva os problemas abaixo usando a linguagem de programação Python, os conceitos da disciplina Lógica de Programação e os conceitos da disciplina Programação Orientada a Objetos.


- Implemente:
1.	Pense numa entidade para desenvolver uma classe com o método construtor e pelo menos quatro atributos, sendo pelo menos um atributo string, pelo menos um atributo float e pelo menos um atributo int.

2.	Crie os métodos gets (consultar) de todos os atributos.

3.	Crie os métodos sets (alterar) de todos os atributos.

4.	Faça uma crítica em um método set numérico para evitar dados inconsistentes, por exemplo, para evitar erro de entrada de dados. 

5.	Crie o método mostra dados 1 usando os atributos da classe. 

6.	Crie o método mostra dados 2 usando os métodos get da classe.

7.	Crie o método retorna dados para retornar todos os atributos da classe.

8.	Crie um método para aumentar o valor numérico de um dos atributos numéricos.

9.	Peça para o usuário fornecer (digitar) o valor do aumento do veículo.

10.	Use sua criatividade, elabore o enunciado (o problema) e implemente mais um método funcional neste sistema.

11.	No método main, teste (use) a classe criada, crie (instancie) pelo menos quatro objetos da classe e use (chame) todos os métodos desenvolvidos na classe.


Obs.: não resolva os exercícios iguais aos desenvolvidos nas aulas, não use a mesma classe da aula.
