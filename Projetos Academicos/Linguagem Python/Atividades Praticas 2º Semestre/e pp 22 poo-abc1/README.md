- Resolva os problemas usando a linguagem de programação Python, os conceitos da disciplina Lógica de Programação, os conceitos da disciplina Programação Orientada a Objetos (POO) com herança (inheritance) e classe abstrata (ABC).

- Implemente:
1.	Elabore um problema que envolva pelo menos três entidades para implementar pelo menos três classes (uma superclasses e duas subclasses). Implemente as classes num arquivo separado do main.
2.	Implemente a superclasse abstrata que herda da classe ABC do Python.
3.	Implemente o método construtor com pelo menos três atributos comuns e pelo menos um atributo default (padrão).
4.	Crie os métodos (defs) gets (consultar) de todos os atributos da superclasse.
5.	Crie os métodos (defs) sets (alterar) de todos os atributos da superclasse. Faça uma crítica (validação) em pelo menos um dos métodos sets da superclasse.
6.	A superclasse abstrata deve ter pelo menos um método abstrato.
7.	Use sua criatividade e elabore pelo menos dois métodos funcionais na superclasse, ou seja, mais dois métodos além dos métodos gets e sets. Esses métodos funcionais podem ser usados pelos objetos das subclasses.

8.	Implemente pelo menos duas subclasses que herdam da superclasse abstrata do item dois usando os conceitos de herança e classe abstrata (ABC).
9.	Crie os respectivos métodos construtores de cada subclasse com pelo menos quatro atributos e pelo menos um atributo com valor default (padrão).
10.	Crie os métodos (defs) gets (consultar) de todos os atributos específicos das subclasses.
11.	Crie os métodos (defs) sets (alterar) de todos os atributos específicos das subclasses. Faça pelo menos uma crítica (validação) em um dos métodos sets das subclasses.
12.	Nas duas subclasses concretas, sobrescreva o método abstrato da superclasse abstrata e crie pelo menos mais um método concreto funcional

13.	Implemente o método main num segundo arquivo.py, separado das classes. Sintaxe: 
from nome_arquivo_sem_extensao import NomeClasse1, NomeClasse2, ..., NomeClasseN
14.	No método main, teste (use) todas as classes criadas, crie (instancie) pelo menos um objeto da superclasse e use (chame) todos os métodos desenvolvidos nas superclasses. 
15.	No método main, crie (instancie) pelo menos dois objetos de cada subclasse concreta, o primeiro objeto com todos os argumentos e o segundo objeto com três argumentos. E use (chame) todos os métodos desenvolvidos nas classes (superclasse e subclasses).
