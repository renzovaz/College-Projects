Exemplo 18 - Revisao sobre Bootstrap

O que e o Bootstrap

O Bootstrap é o framework front-end mais popular do mundo, utilizado para criar sites e aplicações web responsivos de forma rápida e fácil. Ele fornece uma coleção de componentes CSS, HTML e JavaScript pré-estilizados (como botões, formulários e navegação) que se adaptam automaticamente a diferentes tamanhos de tela (mobile-first).

Bootstrap Cores Contextuais

As cores contextuais no Bootstrap 5 são classes utilitárias baseadas em temas que permitem transmitir significado através das cores, facilitando a estilização de textos, fundos, bordas e componentes. Elas seguem um padrão de nomenclatura (primary, success, danger, etc.) que pode ser aplicado a diferentes elementos

Bootstrap Container e container-fluid

Em Bootstrap, as classes .container e .container-fluid são elementos de layout fundamentais, mas com comportamentos de largura distintos.

Bootstrap container

O .container cria um layout de largura fixa responsiva (ou "boxed layout").

  • Comportamento: Ele define uma max-width em cada breakpoint responsivo (pequeno, médio, grande, etc.), ajustando seu tamanho em "degraus" à medida que a largura da tela muda.
  • Alinhamento: O conteúdo dentro do container é automaticamente centralizado na página, com margens laterais (padding) para evitar que o conteúdo toque as bordas da tela.
  • Ideal para: Layouts de conteúdo principal onde se deseja uma leitura confortável e centralizada, como artigos de blog ou seções de texto.

Uso do Container-Fluid

A classe .container-fluid no Bootstrap cria um contêiner de largura total (100%), abrangendo toda a extensão da tela ou dispositivo. Diferente do .container fixo, ele não possui larguras máximas definidas, sendo ideal para layouts responsivos que ocupam a tela inteira, como headera, footers ou seções.
