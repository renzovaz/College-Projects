Exemplo 5 HTML - Uso das tags de Imagem e video

Este é um exemplo de uso das tags de imagem e vídeo em HTML. A tag de imagem é usada para inserir uma imagem em uma página web, enquanto a tag de vídeo é usada para incorporar um vídeo. Ambas as tags possuem atributos que permitem personalizar a exibição do conteúdo, como largura, altura e controles de reprodução. O uso adequado dessas tags pode melhorar a experiência do usuário e tornar o conteúdo mais atraente visualmente.

Este exemplo mostra a diferentes formas de usas as tags <img> e <video>, para inserir imagens e vídeos em uma página web, alem disso tabem vamos usas a tag figCaption e Figura que tem como objetivo de agrupar e colocar uma legenda na imagem e ou video.
