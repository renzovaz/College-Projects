Exemplo 5 HTML - Uso das tags de Imagem e vídeo

Este é um exemplo de uso das tags de imagem e vídeo em HTML. A tag de imagem é usada para inserir uma imagem em uma página web, enquanto a tag de vídeo é usada para incorporar um vídeo.

Este exemplo mostra as diferentes formas de usar as tags <img> e <video>. Também usamos as tags <figure> e <figcaption> para agrupar o conteúdo e inserir uma legenda.
