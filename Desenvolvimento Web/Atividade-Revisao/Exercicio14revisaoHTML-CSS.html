Exemplo 14 - HTML CSS - Flexbox

Flex-wrap

flex-wrap:wrap;

A propriedade CSS flex-wrap define se os itens flexíveis são forçados a ficarem na mesma linha ou se podem ser quebrados em várias linhas.

O valor padrão do flex-wrap é nowrap, o que significa que os itens flexíveis são forçados a ficarem na mesma linha, mesmo que isso signifique que eles sejam redimensionados para caber. Se o valor for wrap, os itens flexíveis podem ser quebrados em várias linhas, se necessário.

O flex-wrap também controla a direção em que as linhas são empilhadas. O valor wrap padrão empilha as linhas da esquerda para a direita, se o flex-direction for row, ou de cima para baixo, se o flex-direction for column. O valor wrap-reverse empilha as linhas da direita para a esquerda ou de baixo para cima, respectivamente.
