HTML é uma linguagem de marcação usada para criar páginas web. Ela é composta por uma série de elementos que definem a estrutura e o conteúdo de uma página. O HTML e fundamental para o desenvolvimento web, permitindo que os desenvolvedores criem páginas interativas e visualmente atraentes. Com o HTML, é possível adicionar texto, imagens, vídeos, links e muito mais, tornando a experiência do usuário mais rica e envolvente.

O HTML é composto por varias tags, cada uma com uma função específica. As tags são utilizadas para estruturar o conteúdo da página, definir elementos como títulos, parágrafos, links, imagens e vídeos. Nessa Revisão vamos estudar as principais tags.

Neste exemplo, estamos utilizando a tag 'p', para criar parágrafos de texto.
