Exemplo 6 HTML - Tabelas

Descrição

Este exemplo mostra como podemos usar a tag <table> para criar tabelas. Tabelas devem ser utilizadas para mostrar dados tabulares, nunca para outra finalidade.

  • <table> - Define a tabela.
  • <tr> - Define uma linha da tabela (table row).
  • <th> - Define o cabeçalho da tabela (table header).
  • <td> - Define o conteúdo/célula da tabela (table data).
