CSS

O CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é uma linguagem usada para estilizar elementos em páginas web. Ele permite controlar a aparência de um site, como cores, fontes, layouts, espaçamentos e muito mais, separando o conteúdo (HTML) da apresentação visual.

Para aplicarmos as folhas de estilo a um documento html, devemos usar regras que são expostas por meio de seletores, propriedades e valores. A seguir descrevemos os pricipais seletores utilizados.

Princípais tipos de seletores CSS

  • Seletor de Tipo (tag) Esse seletor é utilizado por meio de tags htmls. Ou seja, ele estiliza elementos com base na tag html.
  • Seletor de Id Esse seletor estiliza um elemento html que tenha o atributo id especificado. Normalmente ele é único.
  • Seletor de Classe Esse seletor estiliza elementos por meio do atributo class, ou seja, ele estiliza elementos com uma classe específica.

Nesta página exemplificamos os elementos ou tags semânticas e aplicamos o css para estilizar e demostrar sua utilização por meio de seletores diversos.

Para usarmos o css nesta página vamos usar a tag style e dentro dela colocaremos nossas regras css.
