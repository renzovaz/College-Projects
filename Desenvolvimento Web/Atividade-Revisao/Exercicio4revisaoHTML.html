Exemplo 4 HTML - Elementos HTML (listas ordenadas e listas não ordenadas)

Listas

Listas em HTML são representadas pelas tags <ul> (lista não ordenada) e <ol> (lista ordenada). Essas listas possuem diversas finalidades, podendo ser utilizadas para exibir informacoes ou simplesemente serem utilizadas em menus.

Toda vez que usarmos listas ordenadas ou não ordenadas, devemos utilizar a tag <li> para definir cada item da lista.

Exemplo de Lista Ordenada

  1. Item de Lista 1
  2. Item de Lista 2
    1. SubItem 2.1
    2. SubItem 2.2
    3. SubItem 2.3
    4. SubItem 2.4
  3. Item de Lista 3
  4. Item de Lista 4
  5. Item de Lista 5
    1. SubItem 5.1
    2. SubItem 5.2
    3. SubItem 5.3
      1. SubItem 5.3.1
      2. SubItem 5.3.2
      3. SubItem 5.3.3
      4. SubItem 5.3.4
    4. SubItem 5.4

      • SubItem 5.3.2
