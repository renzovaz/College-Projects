O que é o Grid Layout?

O grid layout é bidmensional, permitindo qie o desenvolvedor possa separar o layout da página em linhas e colunas, diferentemente do flexbox que é unidmensional, ordenando os elementos em linha.

Flexgrid ou Grid Layout

O Flexbox e Grid Layout podem ser usados em conjunto.Você pode aplicar o Flexbox a elementos dentro de um layout de grad (grid layout), aproveitando o melhor dos dois mundos.

A escolha entre Flexbox e Grid Layout não se trata de um substituir outro, mas sim de entender as necessidades específicas do layout e escolher a técnica que melhor atende a esses necessidades.
