Exemplo 12 HTML CSS - Flexbox

O Flexbox é um dos sistemas mais importantes para criar layouts responsivos modernos. Ele permite que criemos layouts complexos de forma simples.

Para utilizá-lo precisamos usar a propriedade display, com a opção flex. Neste exemplo usamos alguns recursos fundamentais da sua estrutura. Mostraremos a utilização das pripriedades flex direction que possui propriedades que permitem que um elemento possa ser organizado no sentido horizontal e ou vertical.
